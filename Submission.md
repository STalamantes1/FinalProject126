# Final Project Submission

For your final project, you will create a Python program of your choice, upload it to GitHub, and submit the public GitHub repository link here on Canvas.

The full final project instructions, required files, and grading expectations are posted in the class GitHub assignment README.

Review the README carefully before starting and before submitting.

## Submission

Submit your public GitHub repository link in Canvas.

Make sure your repository includes:

- `starter_code.py` IF using claude 
- `final_code.py`
- `specs.md` If using Claude 
- `README.md`
- `Write-Up`
- any additional files needed for your project

Your repository must be public so I can view and grade it.
